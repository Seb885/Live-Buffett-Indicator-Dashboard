# Live Buffett Indicator Dashboard

This repository hosts a live, interactive dashboard tracking the **Warren Buffett Indicator** for the US Market.

## About the Indicator
The Buffett Indicator is calculated as:
**Total US Stock Market Capitalization / US Nominal GDP**

It is a broad measure of whether the market is overvalued, fairly valued, or undervalued relative to the size of the economy.

## Live Dashboard
**View the live dashboard here: [https://seb885.github.io/Live-Buffett-Indicator-Dashboard/](https://seb885.github.io/Live-Buffett-Indicator-Dashboard/)**

## Data Sources
- **Market Cap**: Wilshire 5000 Total Market Index (via Yahoo Finance).
- - **GDP**: US Nominal GDP (via FRED).
 
  - ## Technology
  - - **Frontend**: Vanilla HTML/CSS/JS with Plotly.js for visualizations.
